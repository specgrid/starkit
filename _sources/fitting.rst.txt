*******
Fitting
*******

There are various external packages that are used for fitting. Here are the
instructions to install some of them.

MultiNest
---------

When using MultiNest please make sure to cite their works (arXiv:0809.3437,
arXiv:0704.3704, arXiv:1306.2144) as well as the wrapper PyMultiNest
( `Buchner et al. 2014, A&A <http://www.aanda.org/articles/aa/abs/2014/04/aa22971-13/aa22971-13.html>`_)

To install the software please go to `http://johannesbuchner.github.io/PyMultiNest/ <http://johannesbuchner.github.io/PyMultiNest/>`_
